# 01 · 化工板块：品种解析与产业链地图

> **一句话总结**：你关注的 9 个代码，是中国能源化工里**三条聚合物产业链**的核心——聚酯链(PX→PTA+EG)、烯烃链(MA/PE/PP/PVC)、苯链(BZ→EB)；而且**9 个全部既有期货又有期权**。化工研究的灵魂不是单看一个品种，而是看**产业链上下游的利润/价差怎么分配**。
>
> **关键问题**：① 这 9 个品种分别在产业链的哪个位置？ ② 为什么化工要"成链"地看，而不是孤立地看一个品种？ ③ 你的清单覆盖全了吗，缺什么？

## 二、产业链地图：三条主线

化工品几乎都从**原油/煤/丙烷**这几个源头出发，逐级加工。你的 9 个品种正好串成三条链：

### 主线 1 · 聚酯链（芳烃链）—— "做化纤和塑料瓶的"
```
原油 → 石脑油 → 重整 → PX(对二甲苯) ──氧化──→ PTA ┐
                                                    ├──→ 聚酯 PET → 涤纶短纤 / 瓶片
              乙烯 ──氧化──→ EG(乙二醇/MEG) ────────┘
```
- 配方记忆：**聚酯 ≈ 0.86×PTA + 0.34×EG**（约 PTA 占 6 成、EG 占 4 成）。
- 这是中国**最成熟、上下游期货最齐全**的一条链：PX、PTA、EG 三个原料端全有期货期权，终端短纤(PF)、瓶片(PR)也有期货。
- 终端需求 = 纺织服装（涤纶）+ 饮料包装（瓶片），强出口属性。

### 主线 2 · 烯烃链（聚烯烃）—— "做塑料制品的"
```
          ┌─ 油头：原油 → 石脑油 → 裂解 → 乙烯 / 丙烯
原料三路 ─┼─ 煤头：煤 → 甲醇(MA) → MTO/MTP → 乙烯 / 丙烯
          └─ 气头：丙烷(LPG) → PDH → 丙烯
乙烯 → 聚乙烯 PE(L)
丙烯 → 聚丙烯 PP
乙烯/电石(煤) → PVC(V)
```
- **甲醇 MA 是这条链的枢纽**：它既是独立品种，又是"煤制烯烃"的核心中间体——所以甲醇价格和 PP/PE 利润高度联动。
- **中国特色**：同一个聚烯烃，可以来自油（油头）、煤（煤头）、气（PDH），三种成本路径并存，谁便宜谁开工——这造就了化工独有的"成本路径切换"逻辑。
- PVC 偏氯碱体系（电石法/乙烯法），和烧碱是"联产兄弟"。

### 主线 3 · 苯链（苯乙烯链）—— 最新成型的一条
```
原油/重整 → 纯苯(BZ) ──烷基化/脱氢──→ 苯乙烯(EB) → PS / EPS / ABS（家电、泡沫、塑料）
```
- 2025 年 7 月纯苯上市后，"纯苯→苯乙烯"上下游齐全，可以做**加工利润套利**了。

## 三、你的清单全面吗？覆盖了什么，缺了什么

**覆盖的**：三条聚合物主线的**核心原料 + 终端聚合物**，这是能源化工里"合成材料"最主干的部分。切入点选得非常好、非常聚焦。

**缺的（按重要性）**：
| 缺口 | 品种 | 为什么值得补 |
|---|---|---|
| 聚酯下游终端 | 短纤 PF、瓶片 PR（郑商所，均有期货） | 补全聚酯链终端，能算"聚酯全链利润" |
| 氯碱配套 | 烧碱 SH（郑商所，期货+期权） | 和 PVC 联产，氯碱平衡是 PVC 供给的关键变量 |
| 煤化工兄弟 | 尿素 UR（郑商所，期货+期权） | 和甲醇同属煤化工，争夺合成气/煤炭 |
| 上游能源（成本端） | 原油 SC、LPG(PG)、沥青 BU、燃料油 FU/LU | 化工成本的"总指挥"，看化工必看油 |
| 无机/建材 | 纯碱 SA、玻璃 FG | 偏建材，和你的聚合物链关系较远，可暂缓 |
| 合成橡胶 | 丁二烯橡胶 BR、天然橡胶 RU、20号胶 NR | 橡胶系自成体系，可作为独立板块 |

> 结论：你的清单 = **"聚合物三条链核心"**，作为化工入门主轴完全够用。下一步最该补的是**上游原油（成本锚）**和**聚酯下游 PF/PR（算全链利润）**。

## 四、按框架的初步分析（化工板块的研究骨架）

### L2 供需框架：化工是典型的「产能周期」品种
- **供给端**看三件事：① 新装置投产/产能投放节奏（化工最重要的供给变量）；② 开工率（受检修、利润、限电影响）；③ 进口。
- **需求端**看：下游开工率 → 终端（地产、家电、纺服、出口）→ 季节性。
- **库存**：工厂库存 + 社会库存 + 港口库存（如 EG、甲醇看港口）。

### L3 价格的语言：化工的灵魂是「价差/加工利润」
化工研究**很少孤立看单价**，而是看**产业链每个环节的加工利润**（上下游价差）：
- **甲醇制烯烃利润** ≈ PP/PE 价格 − 3×甲醇（约 3 吨甲醇产 1 吨烯烃）
- **PTA 加工费** ≈ PTA − 0.66×PX
- **苯乙烯利润** ≈ EB − 纯苯 − 乙烯成本
- **聚酯利润** ≈ 聚酯 − 0.86×PTA − 0.34×EG

> 谁的利润高，谁就有动力多开工→增加供给→利润回归。**利润就是供给的调节阀**。这是化工最核心的交易逻辑。

### L4 驱动因素
- **原油**：成本端总指挥，几乎所有化工品的成本锚。
- **装置动态**：投产、检修、意外停车（化工特有的高频供给冲击）。
- **成本路径切换**：油头/煤头/气头谁便宜（看油价 vs 煤价 vs 丙烷）。
- **终端需求**：地产（PVC、PP管材）、出口（聚酯）、家电（苯乙烯/ABS）。
- **季节性**：农膜（PE，秋季）、检修季（春秋）。

### L5 数据源（化工专用）
- 隆众资讯(oilchem)、卓创资讯、钢联Mysteel、CCF（聚酯专业）。
- 交易所库存/仓单、港口库存（EG/甲醇）、装置开工率周报。

### L6 决策方式：化工最经典的是「价差/利润套利」
- 单边（赌某品种供需）门槛高、风险大；
- 化工更多做**产业链套利**：做多/做空某个加工利润、跨期（近远月）、内外盘（PX/PTA 内外价差）。
- 有了期权后，还能用期权表达"利润区间""波动率"观点。

## 五、建议切入点

1. **先吃透「聚酯链」**：PX→PTA+EG 上下游期货最全、逻辑最清晰，是练"产业链利润"思维的最佳样本。
2. 再补**原油**这个成本锚，理解化工的"成本端"。
3. 然后看**烯烃链的成本路径切换**（油头/煤头/气头），体会化工独有的供给弹性。
